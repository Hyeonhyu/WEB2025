Chapter 17 요약

1. API란?

Application Programming Interface의 약자로, 프로그램 간에 기능이나 데이터를 주고받기 위한 인터페이스입니다.

2. 웹스토리지 API

브라우저에 데이터를 저장하는 API로, 크게 두 가지가 있습니다.

  • localStorage: 영구 저장, 브라우저를 닫아도 데이터 유지
  • sessionStorage: 세션 저장, 브라우저 탭을 닫으면 데이터 삭제

localStorage 예제


// 데이터 저장
localStorage.setItem('username', 'hyeonhyu');

// 데이터 읽기
const name = localStorage.getItem('username');
console.log(name);  // hyeonhyu

// 데이터 삭제
localStorage.removeItem('username');

// 전체 삭제
localStorage.clear();

sessionStorage 예제


// 데이터 저장
sessionStorage.setItem('tempData', '12345');

// 데이터 읽기
const temp = sessionStorage.getItem('tempData');
console.log(temp);  // 12345
