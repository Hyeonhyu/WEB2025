Chapter 13 요약

1. 비동기 처리 방식

자바스크립트는 기본적으로 싱글 스레드이지만, 비동기 처리를 통해 동시에 여러 작업을 수행 가능

  • 콜백 함수
  • 프로미스 (Promise)
  • async/await

2. 프로미스 (Promise)

비동기 작업의 성공/실패 결과를 나타내는 객체


const promise = new Promise((resolve, reject) => {
  const success = true;
  if (success) {
    resolve('성공!');
  } else {
    reject('실패...');
  }
});

promise
  .then(result => console.log(result))  // 성공!
  .catch(error => console.error(error));

3. fetch API

비동기 HTTP 요청을 위한 최신 API


fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('에러:', error));

4. async 함수와 await 예약어

프로미스를 좀 더 직관적으로 다룰 수 있게 하는 문법


async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    if (!response.ok) throw new Error('네트워크 문제');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('에러:', error);
  }
}

fetchData();
← 홈으로 돌아가기
