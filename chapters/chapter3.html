Chapter 3 요약

1. 연산자

  • 산술 연산자: +, -, *, /, %
  • 문자열 연결 연산자: + (예: "Hello" + "World")
  • 증감 연산자: ++, --
  • 대입 연산자: =, +=, -=, *=, /=
  • 비교 연산자: ==, ===, !=, !==, >, <, >=, <=
  • 논리 연산자: && (AND), || (OR), ! (NOT)

2. 조건문

if / else if / else


let score = 75;

if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else if (score >= 70) {
  console.log("C");
} else {
  console.log("F");
}

삼항 연산자

형식: 조건 ? 참일 때 값 : 거짓일 때 값


let age = 20;
let result = (age >= 19) ? "성인" : "미성년자";
console.log(result);

switch 문


let day = "화요일";

switch (day) {
  case "월요일":
    console.log("Monday");
    break;
  case "화요일":
    console.log("Tuesday");
    break;
  default:
    console.log("기타 요일");
}

3. 반복문

for 문


for (let i = 1; i <= 5; i++) {
  console.log("i =", i);
}

while 문


let i = 1;
while (i <= 5) {
  console.log("i =", i);
  i++;
}

do...while 문


let j = 1;
do {
  console.log("j =", j);
  j++;
} while (j <= 5);

break / continue


for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // 3 건너뜀
  if (i === 5) break;    // 5에서 종료
  console.log(i);
}
