Chapter 6 요약

1. 이벤트 처리

사용자의 동작(클릭, 입력 등)에 반응하는 기능.

  • onclick 등의 HTML 속성 사용
  • addEventListener() 메서드 권장

// 방법 1: HTML 속성
<button onclick="sayHi()">클릭</button>

// 방법 2: JS에서 addEventListener
document.querySelector("button").addEventListener("click", sayHi);

function sayHi() {
  alert("안녕하세요!");
}

2. 이벤트 객체

이벤트 발생 시 자동으로 전달되는 정보 객체 (event 또는 e)

  • event.target: 이벤트가 발생한 실제 요소
  • event.type: 이벤트 종류 (예: click)
  • event.preventDefault(): 기본 동작 취소

const link = document.querySelector("a");
link.addEventListener("click", function(e) {
  e.preventDefault(); // 링크 이동 막기
  console.log("클릭한 요소:", e.target);
  console.log("이벤트 타입:", e.type);
});

3. 이벤트 전파 (Event Propagation)

이벤트가 발생했을 때, DOM 트리를 따라 전달되는 방식.

① 버블링 (기본 동작)

이벤트가 안쪽 요소에서 바깥쪽으로 전달됨.

② 캡처링

이벤트가 바깥쪽 요소에서 안쪽으로 전달됨 (옵션 필요).


document.getElementById("outer").addEventListener("click", function() {
  alert("바깥쪽 div");
});

document.getElementById("inner").addEventListener("click", function(e) {
  alert("안쪽 div");
  e.stopPropagation(); // 이벤트 전파 중지
});

e.stopPropagation()으로 전파를 막을 수 있음.

예시 구조


<div id="outer">
  <div id="inner">
    클릭하세요
  </div>
</div>
