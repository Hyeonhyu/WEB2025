Chapter 18 요약

1. 공개 API란?

누구나 사용할 수 있도록 공개된 API로, 주로 웹 서비스나 데이터를 외부에 제공하기 위해 만듭니다.

2. 공개 API 활용 예

  • 날씨 정보 API (예: OpenWeatherMap)
  • 지도 API (예: 카카오, 구글 지도 API)
  • 뉴스, 영화, 교통 정보 API

3. 공개 API 사용 방법

  1. API 제공처에서 API 키 발급 받기
  2. 문서(Documentation)를 참고하여 요청 형식 확인
  3. fetch, axios 등으로 HTTP 요청 보내기
  4. 응답 데이터를 파싱하여 화면에 표시

간단한 fetch 예제


fetch('https://api.example.com/data?apikey=YOUR_API_KEY')
  .then(response => response.json())
  .then(data => {
    console.log(data);
    // 데이터를 화면에 표시하는 코드 작성
  })
  .catch(error => console.error('Error:', error));
