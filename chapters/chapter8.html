Chapter 8 요약

1. 웹브라우저의 최상위 객체: window

모든 브라우저 객체의 최상위. 전역 변수나 함수도 window의 속성으로 존재.

  • window.alert(), window.setTimeout(), window.location 등
  • window는 생략 가능: alert() ≡ window.alert()

alert("Hello!"); // window.alert와 동일
console.log(window.innerWidth); // 브라우저 창 너비

2. Date 객체

현재 시간, 날짜 관련 작업에 사용


const now = new Date();         // 현재 날짜와 시간
const birthday = new Date(2001, 6, 18); // 월은 0부터 시작 (7월은 6)
console.log(now.getFullYear()); // 연도
console.log(now.getMonth() + 1); // 월
console.log(now.getDate());     // 일

자주 쓰는 메서드

  • getFullYear(), getMonth(), getDate()
  • getDay(): 요일 (0 = 일요일)
  • toLocaleString(): 지역 기준 날짜/시간 문자열

3. Math 객체

수학 관련 계산을 위한 내장 객체. 직접 생성하지 않고 바로 사용.


console.log(Math.PI);           // 원주율
console.log(Math.round(4.7));   // 반올림 → 5
console.log(Math.floor(4.9));   // 내림 → 4
console.log(Math.ceil(4.1));    // 올림 → 5
console.log(Math.random());     // 0~1 사이의 난수
console.log(Math.max(1, 5, 3)); // 최댓값 → 5

자주 쓰는 메서드

  • Math.random(): 0 이상 1 미만 난수
  • Math.floor(): 소수점 내림
  • Math.ceil(): 소수점 올림
  • Math.round(): 반올림
  • Math.max(), Math.min(): 최대/최소값
