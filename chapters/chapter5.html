Chapter 5 요약

1. DOM (Document Object Model)

HTML 문서를 자바스크립트에서 제어할 수 있도록 객체 모델로 표현한 것.

각 HTML 요소는 노드(node)로 표현되며, 계층적으로 구성됨.

2. DOM 트리 구조

HTML 문서는 트리 형태로 구성되어 있으며, 부모-자식 관계로 연결됨.

  • 문서 루트: document
  • 노드 타입: 요소 노드, 텍스트 노드, 속성 노드 등

3. 웹 요소 접근 방법

  • document.getElementById("id")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")
  • document.querySelector("선택자")
  • document.querySelectorAll("선택자")

const title = document.getElementById("mainTitle");
title.textContent = "제목 바뀜!";

4. 자바스크립트로 스타일 수정하기

style 속성을 통해 CSS 스타일을 변경할 수 있음.


const box = document.querySelector(".box");
box.style.backgroundColor = "lightblue";
box.style.fontSize = "20px";

5. DOM에서 폼 다루기

  • 폼 요소 접근: document.forms["formName"] 또는 document.querySelector("form")
  • 입력값 가져오기: input.value
  • 체크 여부: checkbox.checked

const form = document.querySelector("#loginForm");
const userId = document.querySelector("#userId");
const remember = document.querySelector("#remember");

form.addEventListener("submit", function(event) {
  event.preventDefault(); // 제출 막기
  alert("ID: " + userId.value + ", 저장: " + remember.checked);
});
