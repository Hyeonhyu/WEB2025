Chapter 4 요약

1. 함수(Function)

자주 사용하는 코드 블록을 재사용하기 위한 구조입니다.


function sayHello(name) {
  console.log("안녕하세요, " + name + "님!");
}

sayHello("현휴");

2. 스코프(Scope)

변수가 유효한 범위를 의미하며, 전역 변수와 지역 변수로 구분됩니다.


let x = 10; // 전역 변수

function testScope() {
  let y = 20; // 지역 변수
  console.log(x); // 가능
  console.log(y); // 가능
}

console.log(x); // 가능
console.log(y); // 오류! y는 함수 안에서만 사용 가능

3. 함수 표현식 (익명 함수)

함수를 변수에 저장해서 사용하는 방식입니다. 보통 이름 없이 익명으로 작성합니다.


const greet = function(name) {
  console.log("Hi, " + name);
};

greet("모카룡");

4. 화살표 함수

간단하게 함수를 작성할 수 있는 문법입니다.


const add = (a, b) => a + b;

console.log(add(3, 5)); // 8

5. 전개 구문(Spread Syntax)

...을 이용해 배열이나 객체의 값을 펼치거나 복사할 수 있습니다.


// 배열 복사
const arr1 = [1, 2, 3];
const arr2 = [...arr1];
console.log(arr2); // [1, 2, 3]

// 배열 결합
const combined = [...arr1, 4, 5];
console.log(combined); // [1, 2, 3, 4, 5]

6. 타이머 함수

일정 시간 후에 코드를 실행하거나 반복 실행합니다.

setTimeout()


setTimeout(() => {
  console.log("3초 후에 실행됩니다.");
}, 3000);

setInterval()


let count = 0;
const intervalId = setInterval(() => {
  console.log("1초마다 실행: " + ++count);
  if (count === 5) clearInterval(intervalId);
}, 1000);
← 홈으로 돌아가기
