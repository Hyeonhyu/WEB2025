1. 그래픽 요소 변형 (Transformations)

  • translate(x, y): 좌표 이동
  • rotate(angle): 회전 (라디안 단위)
  • scale(sx, sy): 크기 변경
  • transform(a, b, c, d, e, f): 복합 변형 매트릭스 적용
  • setTransform(a, b, c, d, e, f): 변형 매트릭스 초기화 후 설정

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

ctx.fillStyle = 'blue';

// 원래 사각형
ctx.fillRect(20, 20, 100, 50);

// 이동 후 사각형
ctx.translate(150, 0);
ctx.fillStyle = 'red';
ctx.fillRect(20, 20, 100, 50);

// 변형 초기화
ctx.setTransform(1, 0, 0, 1, 0, 0);

2. 그래픽 요소 합성 (Compositing)

캔버스에서 겹치는 영역을 어떻게 처리할지 지정

  • globalAlpha: 투명도 설정 (0.0 ~ 1.0)
  • globalCompositeOperation: 합성 방식 지정

ctx.globalAlpha = 0.5;
ctx.fillStyle = 'green';
ctx.fillRect(50, 50, 100, 100);

ctx.globalCompositeOperation = 'multiply';
ctx.fillStyle = 'yellow';
ctx.fillRect(100, 100, 100, 100);

// 합성 모드 초기화
ctx.globalCompositeOperation = 'source-over';
ctx.globalAlpha = 1.0;
