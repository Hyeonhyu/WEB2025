Chapter 14 요약

1. 캔버스(Canvas) 소개

HTML5에서 그래픽을 그릴 수 있는 요소

<canvas id="myCanvas" width="300" height="150"></canvas>

2. 기본 도형 그리기

2D 컨텍스트를 가져와서 도형을 그림


const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

// 사각형 그리기
ctx.fillStyle = 'skyblue';
ctx.fillRect(20, 20, 150, 100);

// 선 그리기
ctx.beginPath();
ctx.moveTo(20, 20);
ctx.lineTo(170, 120);
ctx.stroke();

3. 텍스트 그리기


ctx.font = '20px Arial';
ctx.fillStyle = 'black';
ctx.fillText('안녕하세요!', 30, 50);

4. 이미지 표시하기


const img = new Image();
img.src = 'path/to/image.png';
img.onload = function() {
  ctx.drawImage(img, 50, 70, 100, 100);
};
