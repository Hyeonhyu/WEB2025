Chapter 12 요약

1. HTTP 통신

  • HTTP: 클라이언트와 서버 간 데이터 주고받는 프로토콜
  • 주요 메서드: GET, POST, PUT, DELETE

2. JSON (JavaScript Object Notation)

  • 서버-클라이언트 간 데이터 교환에 많이 쓰임
  • 객체 형태의 문자열 표현
  • JSON.stringify(): 객체 → JSON 문자열
  • JSON.parse(): JSON 문자열 → 객체

const obj = {name: "현휴", age: 24};
const jsonStr = JSON.stringify(obj);
console.log(jsonStr); // '{"name":"현휴","age":24}'

const parsedObj = JSON.parse(jsonStr);
console.log(parsedObj.name); // 현휴

3. 서버에서 자료 가져오기 (fetch API)

비동기 방식으로 HTTP 요청을 보내고 응답을 받는 최신 방식


fetch('https://api.example.com/data')
  .then(response => {
    if (!response.ok) {
      throw new Error('네트워크 오류');
    }
    return response.json();
  })
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error('오류 발생:', error);
  });

4. 예외 처리 (try...catch)

코드 실행 중 발생 가능한 오류를 잡아내고 처리


try {
  let result = riskyOperation();
  console.log(result);
} catch (error) {
  console.error('에러 발생:', error.message);
} finally {
  console.log('항상 실행되는 코드');
}
