1. 구조 분해 할당 (Destructuring Assignment)

배열이나 객체의 값을 편리하게 변수에 할당하는 문법


// 배열 구조분해
const arr = [1, 2, 3];
const [a, b, c] = arr;
console.log(a, b, c); // 1 2 3

// 객체 구조분해
const obj = {name: "현휴", age: 25};
const {name, age} = obj;
console.log(name, age); // 현휴 25

2. 배열 변형 메서드

  • push(), pop(): 끝에 요소 추가/제거
  • shift(), unshift(): 앞에 요소 제거/추가
  • splice(): 배열 중간에 요소 삽입, 삭제
  • slice(): 배열의 일부를 복사해서 반환
  • concat(): 배열 합치기
  • filter(), map(), reduce(): 함수형 배열 처리

3. Map과 Set

Map: 키와 값을 모두 객체로 저장하는 컬렉션


const map = new Map();
map.set('a', 1);
map.set('b', 2);
console.log(map.get('a')); // 1

Set: 중복 없는 값들의 집합


const set = new Set([1, 2, 2, 3]);
console.log(set); // Set {1, 2, 3}

4. 이터레이터(Iterator)와 제너레이터(Generator)

이터레이터는 컬렉션을 순차적으로 접근할 수 있게 해주는 객체

제너레이터는 이터레이터를 쉽게 만드는 함수


// 제너레이터 함수 예제
function* gen() {
  yield 1;
  yield 2;
  yield 3;
}

const iterator = gen();
console.log(iterator.next()); // {value: 1, done: false}
console.log(iterator.next()); // {value: 2, done: false}
console.log(iterator.next()); // {value: 3, done: false}
console.log(iterator.next()); // {value: undefined, done: true}
