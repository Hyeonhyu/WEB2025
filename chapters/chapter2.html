Chapter 2 요약

1. alert() 함수

alert()는 메시지를 알림 창으로 보여주는 함수입니다.

alert("안녕하세요!");

2. prompt() 함수

prompt()는 사용자에게 입력을 받을 수 있는 창을 띄웁니다.


const name = prompt("이름을 입력하세요:");
alert("안녕하세요, " + name + "님!");

3. confirm() 함수

confirm()은 확인/취소 버튼이 있는 창을 띄우며, 사용자의 응답은 true 또는 false로 반환됩니다.


const result = confirm("정말 삭제하시겠습니까?");
if (result) {
  alert("삭제되었습니다.");
} else {
  alert("취소되었습니다.");
}

4. 자료형

  • 문자형(String): "Hello", 'World'
  • 숫자형(Number): 10, 3.14
  • 논리형(Boolean): true, false
  • undefined: 값이 할당되지 않은 변수
  • null: 의도적으로 "없음"을 나타냄
  • 객체(Object): 배열, 함수, 일반 객체 등
