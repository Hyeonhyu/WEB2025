Chapter 10 요약

1. 문자열 접근

문자열은 배열처럼 인덱스로 접근 가능하며, length 속성 사용 가능


const str = "Hello, World!";
console.log(str[0]);       // H
console.log(str.charAt(1)); // e
console.log(str.length);    // 13

2. 주요 문자열 메서드

  • toUpperCase(), toLowerCase(): 대소문자 변환
  • indexOf(), lastIndexOf(): 위치 찾기
  • slice(start, end): 부분 문자열 추출
  • substring(start, end), substr(start, length)
  • trim(): 공백 제거
  • replace(찾을값, 바꿀값)
  • includes(), startsWith(), endsWith()

const str = " JavaScript ";
console.log(str.trim());            // "JavaScript"
console.log(str.toUpperCase());     // " JAVASCRIPT "
console.log(str.indexOf("S"));      // 5
console.log(str.replace("Script", "Rush")); // " JavaRush "

3. 정규표현식(Regex)으로 문자열 다루기

정규표현식은 패턴 매칭에 사용됨

  • /pattern/ 또는 new RegExp("pattern") 사용
  • test(): 패턴 일치 여부
  • match(): 패턴과 일치하는 값 찾기
  • replace(): 패턴에 맞는 부분 치환

const text = "010-1234-5678";
const regex = /\d{3}-\d{4}-\d{4}/;
console.log(regex.test(text));  // true
console.log(text.match(regex)); // ['010-1234-5678']
console.log(text.replace(regex, "비공개")); // "비공개"

4. 문자열과 배열 간 변환

  • split(구분자): 문자열 → 배열
  • join(구분자): 배열 → 문자열

const csv = "현휴,과기대,웹프로그래밍";
const arr = csv.split(","); // ["현휴", "과기대", "웹프로그래밍"]
console.log(arr.join(" & ")); // "현휴 & 과기대 & 웹프로그래밍"
