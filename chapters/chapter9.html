Chapter 9 요약

1. 생성자 함수로 객체 만들기

생성자 함수는 객체를 만드는 함수이며, new 키워드와 함께 사용


function Person(name, age) {
  this.name = name;
  this.age = age;
  this.sayHello = function() {
    console.log("안녕! 나는 " + this.name);
  };
}

const p1 = new Person("현휴", 24);
p1.sayHello(); // 안녕! 나는 현휴

2. 클래스 문법

ES6 이후 도입된 클래스 문법은 객체지향 방식 구현을 더 직관적으로 만들어줌


class Animal {
  constructor(name) {
    this.name = name;
  }

  speak() {
    console.log(`${this.name}이(가) 소리를 낸다.`);
  }
}

const dog = new Animal("강아지");
dog.speak(); // 강아지이(가) 소리를 낸다.

3. 객체의 키와 값에 접근하기

  • 점 표기법: obj.key
  • 대괄호 표기법: obj["key"] — 변수 키 접근 가능

const user = { name: "현휴", age: 24 };
console.log(user.name);     // 점 표기법
console.log(user["age"]);   // 대괄호 표기법

4. 프로토타입(prototype)과 상속

자바스크립트는 프로토타입 기반 상속을 사용


function Person(name) {
  this.name = name;
}
Person.prototype.sayHello = function() {
  console.log("Hi, I'm " + this.name);
};

const me = new Person("현휴");
me.sayHello(); // Hi, I'm 현휴

5. 클래스에서의 상속 (extends)


class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(`${this.name}이(가) 소리를 냅니다.`);
  }
}

class Dog extends Animal {
  speak() {
    console.log(`${this.name}이(가) 멍멍 짖습니다.`);
  }
}

const myDog = new Dog("시로");
myDog.speak(); // 시로가(이) 멍멍 짖습니다.
