Chapter 7 요약

1. DOM 트리와 노드 리스트

HTML 문서는 트리 구조로 표현되며, 각 요소는 노드(Node)로 구성됨.

  • 요소 노드: 태그 요소 (ex. <div>)
  • 텍스트 노드: 텍스트 내용
  • 속성 노드: 태그에 포함된 속성 정보

여러 개의 요소를 선택할 때 NodeList로 반환됨:


const items = document.querySelectorAll("li");
console.log(items.length); // NodeList 길이

2. 웹 문서에 새로운 노드 추가하기

createElement()와 appendChild() 또는 append() 사용


const ul = document.querySelector("ul");
const newItem = document.createElement("li");
newItem.textContent = "새 항목 추가";
ul.appendChild(newItem);

💡 append()과 appendChild() 차이

  • append(): 텍스트 노드도 추가 가능
  • appendChild(): 요소 노드만 추가 가능

3. 노드 삭제하기

노드 제거는 부모 노드를 통해 removeChild() 또는 remove() 사용


// 방법 1: 부모에서 자식 제거
const ul = document.querySelector("ul");
const firstItem = ul.querySelector("li");
ul.removeChild(firstItem);

// 방법 2: 직접 삭제 (최신 브라우저 지원)
firstItem.remove();
